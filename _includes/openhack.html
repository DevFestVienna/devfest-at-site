OpenHack Results

We had many interesting projects being worked on during our OpenHack. Thanks to all participants!
Experiments with hybrid mobile apps

Mihai Ghete worked on a rudimentary hybrid / cross-platform mobile app that can display and rotate a Leaflet.js-based online map based on the orientation sensor.

Technologies used:

  • Apache Cordova
  • Crosswalk
  • Leaflet.js
  • HTML/JS/CSS
MassKerade

Christoph Pacher worked on MassKerade, an installation that transforms the media facade of a building near by a nodal point of a city into an interactive medium. It offers passers-by a change of perspective, the possibility to perceive themselves as an atom, as a small part of a bigger entity. In contrast to atoms, people in Mass|kerade however have the choice to stand out from the masses.

Technologies used:

  • Kinect V1/V2
  • C++11
  • Cinder
  • OpenCV
Mega metrics

Frantisek Kocun worked on I extendiong a project which used for metrics analytics of the server in his job. This included converting some backend logic from Python to Scala, Websockets-based communication, and JavaScript front-end development.

Technologies used:

  • Python
  • Scala
  • Spray
  • Websockets
  • JavaScript
Musicube

Nikolay Kushin, Alberto Pierini and Lorenz Leutgeb worked on Muiscube, a small cube to control a music stream, by altering the genre dependent on the face on top.
Data is transferred via bluetooth and picked up by node-webkit.

Technologies used:

  • Arduino (accelerometer, Bluetooth module)
  • node.js
  • node-webkit
  • HTML5
  • Lame
zinc

In addition, Lorenz Leutgeb worked on zinc, a Docker container bundling the stable release of Google Chrome and latest release of ChromeDriver. Running on top of Xvfb, accessible via x11vnc, and screen recordings being dumped. Ideal for browser testing.
Data is transferred via bluetooth and picked up by node-webkit.

Technologies used:

  • Docker
  • Google Chrome
  • ChromeDriver
  • Xvfb (X Window System)
  • x11vnc (CNV)
  • avconv (WebM)
Playing around with BOINC and HAMLIB

Gregor Beyerle and Marcel Cimander were trying to build their own BOINC project (a simple distributed application) and did some debugging on components of the HAMLIB (api to control amateur radio hardware). In the end they had a running BOINC server with simple hardly configured components (scheduler, feeder, validator, transitioner, deleter) to which an arbitrary number of clients could connect.

Technologies used:

  • Libraries / Frameworks
  • BOINC
  • HAMLIB
  • Languages
  • C/C++
  • PHP
  • BASH (a bit of bash scripting just for the heck of it)
  • Additional Software
  • Apache Web Server
  • modules (cgi, fcgi, php)
logTank

Sigita Grinfelde and Peter Grman were working on logTank, their upcoming centralised log aggregation and analysis tool. logTank will allow its users to collect, store and analyse data of their websites, applications, servers, .... This way logTank customers will always know, for instance, what are the users of their own application searching for, viewing, analyse what problems or errors their application holds and similar.
During the OpenHack they got the core of logTank running and made some example d3.js charts to show what kind of analysis would be possible with logTank. Examples were - how many 404, 304 and 200 codes are sent from or website, what languages are mainly used for tweets of 4 tags, which logTank was following during the time of the OpenHack, etc.

Technologies used:

  • Fluentd
  • CouchDB
  • JavaScript
  • D3.js
Project Danube

Markus Sabadello was refining and working on a FreedomBox prototype (based on Cubieboard open source hardware) and refining it. Project Danube has for several years contributed to efforts for a more free, decentralized, and private Internet. This includes work on the "Federated Social Web", as well as the "FreedomBox". At the end Markus demoed the FreedomBox prototype and its capabilities

Technologies used:

  • PageKite
  • ownCloud
  • Privoxy
  • Debian
  • MailPile
  • ... and many more, open source & free software components for online freedom
